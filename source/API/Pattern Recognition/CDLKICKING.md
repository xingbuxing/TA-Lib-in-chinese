
# CDLKICKING

中文名称：  反冲形态

用法：二日 K 线模式，与分离线类似，助涨上升或下降趋势

调用方法：signal = talib.CDLKICKING(open, high, low, close)

输出：0代表不符合k线形态，100表示符合形态。

## 指标介绍
两日 K 线为秃线，颜色相反，存在跳空缺口。
