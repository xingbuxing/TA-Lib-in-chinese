# CDLMORNINGDOJISTAR

中文名称：十字启明星

用法：出现十字启明星，说明上升趋势启动

调用方法：talib.CDLMORNINGDOJISTAR(open, high, low, close [, penetration=?])

输出：0代表不符合K线形态，100表示符合形态

## 指标介绍

第一根k线：长阴线

第二根k线：向下跳空十字星

第三根k线：长阳线，实体在在第一个长阴线实体内
