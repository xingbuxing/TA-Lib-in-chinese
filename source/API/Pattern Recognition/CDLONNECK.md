# CDLONNECK

中文名称：好友反攻

用法：总是看跌

调用方法：talib.CDLONNECK(open, high, low, close)

输出：0代表不符合K线形态，-100表示符合形态

## 指标介绍

第一根k线：长阴线

第二根k线：阳线低开，收盘价第一前一天的最低价
