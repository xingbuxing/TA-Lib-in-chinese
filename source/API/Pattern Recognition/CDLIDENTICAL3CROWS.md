# CDLIDENTICAL3CROWS

中文名称： 三胞胎乌鸦

用法：三胞胎乌鸦信号比三只乌鸦信号表现出更多的惊慌卖出,预示着市场趋势有一个更大的反转。

调用方法：signal = talib.CDLIDENTICAL3CROWS(open, high, low, close)

输出：0代表不符合k线形态，-100表示符合形态。

## 指标介绍
1.这三个蜡烛都必须为长的黑色实体，且长度应该大致相等。

2.前一天的市场趋势应该为上涨。

3.这三天中每天的开盘价必须等于前一天的收盘价。

4.这三天中每一天的收盘价的位置必须接近当天的最低价。
