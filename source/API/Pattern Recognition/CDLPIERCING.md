# CDLPIERCING

中文名称：穿刺线

用法：出现穿刺线，后市看涨

调用方法：talib.CDLPIERCING(open, high, low, close)

输出：0代表不符合K线形态，100表示符合形态

## 指标介绍

第一根k线：长阴线

第二根k线：长阳线，开盘低于前一天最低价，且收盘至少在前一天的50%以上
